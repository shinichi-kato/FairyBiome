## 入力ファイルの形式
```json
[
    {
        in:["こんにちは","こんばんは","おはよう"],
        out:["こんにちは！!"]
    },
    {
        in:["元気？"],
        out:["{me}は元気です！"]
    },
]
```

## 出力
``` js
{
    vocab: [] , // 辞書に現れる全ワードのリスト
    tfidf: [] , // TFIDF行列
    index: [],  // tfidfの各行がもとの入力ファイルのどの行に対応するかを格納したリスト
}
```
